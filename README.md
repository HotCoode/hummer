###About
----------
每天我们做的事情很多是重复的，有一些不必要的事情占据了大量的时间，<Strong>Done</Strong>为你记录每天做了什么，帮你留出更多的时间做重要的事。


###Digressions
-------------
这只是一个纠正不良习惯的工具，如果任何事都计划着去做，生活会失去很多乐趣，所以当你认为有一些坏习惯已经影响了正常生活那请使用<Strong>Done</Strong>吧，祝你尽快改掉坏习惯。
